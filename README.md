# CodeAlpha_Calculator

A sleek, retro-inspired web-based Calculator designed with a glowing amber LED display aesthetic. This project was developed as part of my Front-End Development Internship at CodeAlpha.

## 🚀 Features

- **LED Display Design:** Dark panel with a glowing amber (LED-style) result display and a dimmer expression line above it.
- **Full Arithmetic Support:** Addition, subtraction, multiplication, division, and percentage calculations.
- **Live Expression Tracking:** Shows the ongoing calculation (e.g. `12 +`) above the current result.
- **Keyboard Support:** Fully usable with the keyboard — number keys, `+ − * /`, `Enter`, `Escape`, `Backspace`, and `%`.
- **Smart Input Handling:** Clear (AC), backspace, decimal point control, and auto-formatted numbers with comma separators.
- **Responsive Layout:** Clean grid-based key layout that works well on both desktop and mobile screens.

## 🛠️ Tech Stack Used

- **HTML:** Semantic structure for the display and keypad.
- **CSS:** Custom styling with CSS variables, box shadows, and glow effects to create the LED look.
- **JavaScript:** Handles all calculation logic, keyboard events, display formatting, and error handling.
